Vaccination record

During the E-services and Web Programming course of my university studies, I was a part of a group that developed a digital vaccination record website that provided the users with a personal vaccination record. Users can log into their account to see injected vaccinations, which doctor who made the injection and upcoming vaccinations. Doctors can log whenever they inject vaccination into a user through their vaccination agency account on the website.

Developed using: HTML /CSS, PHP, and JavaScript in ASP.NET, designed the relational database using Entity Framework.
Project duration: 4 weeks.

What is this? Lets the user see which vaccination is needed in different parts of the world. A great feature to use before a trip!

What is this? Lets the user see which vaccination agencies are located close.

What is this? This is a user's profile where the personal information easily can be modified by the user.

What is this? Lets the user set reminding notifications for vaccinations that need to be injected in the future. This way vaccinations consisting of more than one shot is never forgotten.
